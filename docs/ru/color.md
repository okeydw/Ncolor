### `VWC` (Стили текста)
| Имя | Код | Описание ANSI |
|-----|-----|---------------|
| `NONE` | `0` | Сброс/отсутствие стиля |
| `BOLD` | `1` | Жирный |
| `DIM` | `2` | Тусклый |
| `UNDERLINED` | `4` | Подчёркнутый |
| `RARE_BLINK` | `5` | Медленное мигание |
| `OFTEN_BLINK` | `6` | Частое мигание |
| `REVERSE` | `7` | Инверсия фона/текста |
| `HIDDEN` | `8` | Скрытый текст |

> ⚠️ Стили можно комбинировать побитовым ИЛИ: `VWC.BOLD | VWC.UNDERLINED` → `5`.  
> Все методы цвета игнорируют обработку, если `Config.color == False`.

## `Color.c()`
Применяет стандартные ANSI-цвета и стили текста.

### Сигнатура
```python
@staticmethod
def c(
    text: str,
    fg: int | None = None,
    bg: int | None = None,
    view: int = 0,
    reset: bool | None = None,
) -> str:
```

### Параметры
| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `text` | `str` | — | Исходная строка. Объекты не типа `str` возвращаются без изменений. |
| `fg` | `int \| None` | `None` | Код цвета текста из `FGC`. |
| `bg` | `int \| None` | `None` | Код цвета фона из `BGC`. |
| `view` | `int` | `0` | Код стиля из `VWC`. |
| `reset` | `bool \| None` | `None` | Принудительно добавляет `\033[0m`. Если `None`, использует `Config.autoreset`. |

### Поведение
- Возвращает исходный `text`, если `view == 0`, `fg is None` и `bg is None`.
- Формирует SGR-последовательность: `\033[<view>;<fg>;<bg>m<text>`. Пропущенные параметры исключаются из строки.
- Суффикс сброса добавляется, если `reset is True` или (`reset is None` и `Config.autoreset is True`).

## `Color.rgb()`
Применяет TrueColor (24-bit RGB) форматирование.

### Сигнатура
```python
@staticmethod
def rgb(
    text: str,
    fg: tuple[int, ...] | list[int] | None = None,
    bg: tuple[int, ...] | list[int] | None = None,
    view: int = 0,
    reset: bool | None = None,
) -> str:
```

### Параметры
| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `text` | `str` | — | Исходная строка. |
| `fg` | `tuple \| list \| None` | `None` | `(r, g, b)` для переднего плана. Отсутствующие индексы заменяются на `0`. Неверные типы тихо игнорируются. |
| `bg` | `tuple \| list \| None` | `None` | `(r, g, b)` для фона. Те же правила резервирования/игнорирования, что и у `fg`. |
| `view` | `int` | `0` | Код стиля из `VWC`. |
| `reset` | `bool \| None` | `None` | Принудительное управление суффиксом сброса. |

### Поведение
- Строит расширенные SGR-коды: `38;2;<r>;<g>;<b>` (текст) и `48;2;<r>;<g>;<b>` (фон).
- Если оба параметра цвета невалидны или `None`, возвращается исходный `text`.
- Логика сброса идентична `Color.c()`.

## `Color.strip_ansi()`
Удаляет все ANSI-escape последовательности из строки.

### Сигнатура
```python
@staticmethod
def strip_ansi(text: str) -> str:
```

### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `text` | `str` | Входная строка. Объекты не типа `str` возвращаются без изменений. |

### Поведение
- Использует скомпилированный regex для поиска всех CSI/OSC/SGR последовательностей.
- Заменяет каждое совпадение на один пробел `" "`.
- Возвращает очищенную строку.
