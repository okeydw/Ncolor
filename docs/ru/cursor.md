# `cursor.md` - Управление курсором и экраном

## Класс `Cursor`
Все координаты начинаются с `1` (`x` = столбец, `y` = строка).  
Поведение методов зависит от двух глобальных флагов:
- `Config.cursor`: Если `False`, все методы мгновенно возвращают `None`, полностью отключая вывод.
- `Config.cursor_write`: Если `True`, методы возвращают ANSI-строку вместо печати. Если `False`, строка сразу выводится в `stdout` с `flush()`.

### Методы перемещения
| Метод | Сигнатура | ANSI-код | Описание |
|-------|-----------|----------|----------|
| `UP` | `UP(n: int = 1) -> str \| None` | `\033[{n}A` | Сдвиг курсора вверх на `n` строк. |
| `DOWN` | `DOWN(n: int = 1) -> str \| None` | `\033[{n}B` | Сдвиг курсора вниз на `n` строк. |
| `FORWARD` | `FORWARD(n: int = 1) -> str \| None` | `\033[{n}C` | Сдвиг курсора вправо на `n` столбцов. |
| `BACK` | `BACK(n: int = 1) -> str \| None` | `\033[{n}D` | Сдвиг курсора влево на `n` столбцов. |
| `NEXT_LINE` | `NEXT_LINE(n: int = 1) -> str \| None` | `\033[{n}E` | Переход в начало следующей строки (`n` раз). |
| `PREV_LINE` | `PREV_LINE(n: int = 1) -> str \| None` | `\033[{n}F` | Переход в начало предыдущей строки (`n` раз). |
| `COLUMN` | `COLUMN(n: int = 1) -> str \| None` | `\033[{n}G` | Перемещение в столбец `n` текущей строки. |
| `POS` | `POS(x: int = 1, y: int = 1) -> str \| None` | `\033[{y};{x}H` | Установка абсолютной позиции `(x, y)`. |
| `POS_ALT` | `POS_ALT(x: int = 1, y: int = 1) -> str \| None` | `\033[{y};{x}f` | Альтернативная установка позиции. Функционально идентична `POS`. |
| `HOME` | `HOME() -> str \| None` | `\033[H` | Перемещение в `(1, 1)`. |

> Все числовые параметры проходят валидацию через внутренний `_clamp()`, гарантируя значение `≥ 1`.

### Очистка и управление состоянием
| Метод | Сигнатура | ANSI-код | Описание |
|-------|-----------|----------|----------|
| `CLEAR` | `CLEAR(mode: int = 2) -> str \| None` | `\033[{mode}J` | Очистка экрана. `mode`: `0` → от курсора до конца, `1` → от начала до курсора, `2` → весь экран, `3` → экран + буфер прокрутки. Диапазон ограничен `[0, 3]`. |
| `CLEAR_LINE` | `CLEAR_LINE(mode: int = 0) -> str \| None` | `\033[{mode}K` | Очистка текущей строки. `mode`: `0` → от курсора до конца, `1` → от начала до курсора, `2` → вся строка. Диапазон `[0, 2]`. |
| `HIDE` | `HIDE() -> str \| None` | `\033[?25l` | Скрытие курсора. |
| `SHOW` | `SHOW() -> str \| None` | `\033[?25h` | Отображение курсора. |
| `SAVE` | `SAVE() -> str \| None` | `\033[s` | Сохранение позиции и состояния курсора. |
| `RESTORE` | `RESTORE() -> str \| None` | `\033[u` | Восстановление ранее сохранённого состояния. |

### Прямой вывод
| Метод | Сигнатура | Описание |
|-------|-----------|----------|
| `write` | `write(*codes: str) -> None` | Объединяет переданные строки через пробел и принудительно выводит в `stdout` с `flush()`. Игнорирует `Config.cursor_write` и `Config.cursor`. Пустые строки пропускаются. |
