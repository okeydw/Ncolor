# `other.md` - Инициализация и конфигурация

## `init()`
Точка входа для глобальной настройки библиотеки. Подготавливает терминал к корректной работе с ANSI-кодами и применяет runtime-параметры.

### Сигнатура
```python
def init(
    autoreset: bool = True,
    cursor_write: bool = False,
    color: bool | None = None,
    cursor: bool | None = None,
    debug: bool | None = None,
) -> None:
```

### Параметры
| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `autoreset` | `bool` | `True` | Автоматически добавляет сброс стилей (`\033[0m`) после форматированного текста. |
| `cursor_write` | `bool` | `False` | Режим вывода `Cursor`: `False` — мгновенная запись в `stdout`, `True` — возврат ANSI-строк для ручной конкатенации. |
| `color` | `bool \| None` | `None` | Глобальный переключатель цветов. `None` сохраняет текущее состояние. |
| `cursor` | `bool \| None` | `None` | Глобальный переключатель управления курсором/экраном. `None` сохраняет текущее состояние. |
| `debug` | `bool \| None` | `None` | Включает/выключает вспомогательные функции модуля `Debug`. `None` сохраняет текущее состояние. |

### Логика работы
1. Обновляет атрибуты глобального объекта `Config`.
2. Если `Config.color` или `Config.cursor` равно `True`, автоматически запускаются процедуры совместимости:
   - `enable_windows_vt()` — активирует поддержку ANSI-последовательностей в консоли Windows.
   - `setup_utf8()` — переконфигурирует `sys.stdout` на кодировку UTF-8.
3. Все исключения платформенной инициализации подавляются для обеспечения отказоустойчивости.

### Примеры
```python
from ncolor import init

# Инициализация по умолчанию
init()

# Отключение цветов и отладки, включение режима сборки строк для курсора
init(color=False, debug=False, cursor_write=True)

# Изменение только autoreset, остальные параметры остаются без изменений
init(autoreset=False)
```

### Глобальный объект `Config`
Все настройки хранятся в `ncolor._config.Config`. Прямая модификация возможна, но не рекомендуется:
```python
from ncolor._config import Config

Config.cursor = False      # Полное отключение модуля Cursor
Config.debug = False       # Отключение отладочных функций
Config.autoreset = False   # Отключение автоматического сброса стилей
```
> ⚠️ Прямое изменение `Config` после вызова `init()` применяется мгновенно, но **не** перезапускает процедуры настройки платформы.
