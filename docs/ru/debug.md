# `debug.md` - Отладка

## Класс `Debug`
Вспомогательные утилиты для визуализации и анализа ANSI-вывода.  
**Важно:** Все методы активно работают только при `Config.debug == True`. В противном случае возвращают `None` или пустой словарь `{}`.

### `Debug.all_colors()`
Извлекает публичные целочисленные константы из указанного класса.

#### Сигнатура
```python
@staticmethod
def all_colors(class_type) -> dict[str, int]:
```

#### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `class_type` | `type` | Класс констант (`FGC`, `BGC` или `VWC`). |

#### Возврат
Словарь `{имя_атрибута: значение}`. Если у объекта нет `__dict__` или `Config.debug == False`, возвращает `{}`.

---

### `Debug.test_rgb_color()`
Генерирует тестовую строку для визуальной проверки RGB-фона.

#### Сигнатура
```python
@staticmethod
def test_rgb_color(r: int, g: int, b: int) -> str | None:
```

#### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `r` | `int` | Красный (0–255) |
| `g` | `int` | Зелёный (0–255) |
| `b` | `int` | Синий (0–255) |

#### Возврат
Строку формата `\033[48;2;<r>;<g>;<b>m\n\033[0m` при включённой отладке, иначе `None`.

---

### `Debug.all_ansi()`
Заменяет управляющие символы на читаемые теги для отладки консольного вывода.

#### Сигнатура
```python
@staticmethod
def all_ansi(text: str) -> str | None:
```

#### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `text` | `str` | Входная строка с возможными ANSI-кодами. |

#### Возврат
Строку с заменами: `\x1b`/`\033` → `<ESC>`, `\n` → `<LF>`, `\r` → `<CR>`, `\t` → `<TAB>`. При `Config.debug == False` возвращает `None`.
